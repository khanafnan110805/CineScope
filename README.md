# 🎬 CineScope

CineScope is a modern cinematic movie discovery web application built using HTML, CSS, and JavaScript. The application allows users to search movies in real-time using the OMDb API and view detailed movie information including posters, genres, ratings, actors, and plot summaries through a premium streaming-platform inspired UI.

---

## 🚀 Live Demo

https://cinescope-ur-movies.netlify.app/

---

## ✨ Features

- 🔍 Real-time movie search using OMDb API
- 🎥 Cinematic dark-themed UI
- ⭐ Movie ratings and details
- 🖼️ Dynamic movie posters
- 📱 Fully responsive design
- ⚡ Fast API fetching using Async/Await
- 🎬 Premium streaming-platform inspired layout
